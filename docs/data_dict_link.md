| BẢNG: FACT_LINK                                                                                                                                                                             |              |                                                                     |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------|---------------------------------------------------------------------|
| Bảng sự kiện lưu thông tin từng link sản phẩm (item) mà khách đặt mua. Mỗi bản ghi là một sản phẩm riêng lẻ trong đơn hàng, bao gồm giá, loại hàng và thông tin Cohort phân tích retention. |              |                                                                     |
|                                                                                                                                                                                             |              |                                                                     |
| Tên Cột                                                                                                                                                                                     | Kiểu Dữ Liệu | Mô tả & Ý nghĩa Nghiệp vụ                                           |
| link_id                                                                                                                                                                                     | INT          | Khoá chính - Mã định danh duy nhất của từng link/item sản phẩm      |
| product_name                                                                                                                                                                                | VARCHAR      | Tên sản phẩm (tên gốc từ website nước ngoài, thường bằng tiếng Anh) |
| product_type_name                                                                                                                                                                           | VARCHAR      | Danh mục sản phẩm (khớp với product_type.product_type_name)         |
| name                                                                                                                                                                                        | VARCHAR      | Mã tiền tệ/tuyến của sản phẩm (IDR, JPY, USD,...)                   |
| total_web                                                                                                                                                                                   | FLOAT        | Tổng giá trị sản phẩm trên website gốc (theo tiền tệ nước ngoài)    |
| exchange_rate                                                                                                                                                                               | FLOAT        | Tỷ giá áp dụng khi quy đổi giá sản phẩm sang VNĐ                    |
| final_price_vnd                                                                                                                                                                             | FLOAT        | Giá cuối cùng của sản phẩm quy đổi ra VNĐ                           |
| final_price                                                                                                                                                                                 | FLOAT        | Giá cuối cùng theo tiền tệ gốc (nước ngoài)                         |
| forex_margin                                                                                                                                                                                | FLOAT        | Phí chênh lệch tỷ giá thu từ khách (VNĐ)                            |
| order_id                                                                                                                                                                                    | INT          | Khoá ngoại tới customer_order - Đơn hàng chứa sản phẩm này          |
| created_at                                                                                                                                                                                  | DATETIME     | Thời điểm tạo link/item sản phẩm                                    |
| date                                                                                                                                                                                        | DATE         | Ngày tạo link (đã chuẩn hoá, bỏ giờ)                                |
| customer                                                                                                                                                                                    | INT          | Mã khách hàng đặt link này (khoá ngoại tới customer)                |
| First purchase                                                                                                                                                                              | DATE         | Ngày mua lần đầu tiên của khách hàng                                |
| Cohort Month reve                                                                                                                                                                           | DATE         | Tháng ghi nhận doanh thu trong phân tích Cohort                     |
| Order Month                                                                                                                                                                                 | DATE         | Tháng của đơn hàng (dùng cho phân tích Cohort)                      |
| Cohort Index                                                                                                                                                                                | INT          | Số tháng kể từ lần mua đầu tiên (0 = tháng khởi đầu)                |
