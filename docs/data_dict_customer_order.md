| BẢNG: CUSTOMER_ORDER                                                                                                                                                                        |              |                                                                                                          |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------|----------------------------------------------------------------------------------------------------------|
| Bảng sự kiện (Fact-like) chứa toàn bộ lịch sử đơn hàng của khách hàng. Là bảng trung tâm thể hiện hành vi mua hàng, trạng thái xử lý và các chỉ số RFM/Cohort phục vụ phân tích khách hàng. |              |                                                                                                          |
|                                                                                                                                                                                             |              |                                                                                                          |
| Tên Cột                                                                                                                                                                                     | Kiểu Dữ Liệu | Mô tả & Ý nghĩa Nghiệp vụ                                                                                |
| order_id                                                                                                                                                                                    | INT          | Khoá chính - Mã định danh duy nhất của đơn hàng                                                          |
| ngày_lên_đơn                                                                                                                                                                                | DATETIME     | Thời điểm khách hàng tạo đơn (giờ quốc tế - múi giờ gốc)                                                 |
| customer_id                                                                                                                                                                                 | INT          | Khoá ngoại - Mã định danh khách hàng                                                                     |
| route_id                                                                                                                                                                                    | INT          | Khoá ngoại tới dim_route - Tuyến vận chuyển của đơn hàng                                                 |
| customer_name                                                                                                                                                                               | VARCHAR      | Tên/biệt danh khách hàng (dùng nội bộ)                                                                   |
| order_type                                                                                                                                                                                  | VARCHAR      | Loại dịch vụ: MUA_HO (mua hộ), KY_GUI (ký gửi), CHUYEN_TIEN (chuyển tiền), DAU_GIA (đấu giá)             |
| country                                                                                                                                                                                     | VARCHAR      | Mã tiền tệ nước xuất xứ hàng hoá (IDR, KRW, JPY, USD)                                                    |
| status                                                                                                                                                                                      | VARCHAR      | Trạng thái hiện tại của đơn: DA_GIAO, DA_HUY, CHO_THANH_TOAN, DA_XAC_NHAN, CHO_NHAP_KHO_NN, CHO_GIAO,... |
| price_ship                                                                                                                                                                                  | INT          | Tổng phí vận chuyển của đơn hàng (VNĐ)                                                                   |
| date                                                                                                                                                                                        | DATE         | Ngày tạo đơn (đã chuẩn hoá về date, bỏ giờ)                                                              |
| Recency Days (col)                                                                                                                                                                          | INT          | Số ngày kể từ đơn hàng gần nhất của khách (dùng cho RFM - chỉ số Recency)                                |
| Recency Bucket                                                                                                                                                                              | VARCHAR      | Phân nhóm Recency: 0-7, 8-30, 31-60, >60 ngày                                                            |
| ngày_lên_đơn_VN                                                                                                                                                                             | DATETIME     | Thời điểm tạo đơn quy đổi về múi giờ Việt Nam (GMT+7)                                                    |
| hour                                                                                                                                                                                        | INT          | Giờ tạo đơn (theo giờ VN) - dùng phân tích hành vi theo khung giờ                                        |
| Order Month                                                                                                                                                                                 | DATE         | Tháng của đơn hàng (chuẩn hoá về ngày đầu tháng, dùng cho Cohort)                                        |
| First Order Date                                                                                                                                                                            | DATE         | Ngày đặt đơn đầu tiên của khách hàng                                                                     |
| Cohort Month                                                                                                                                                                                | DATE         | Tháng đặt đơn đầu tiên của khách (cột mốc khởi đầu Cohort)                                               |
| Cohort Index                                                                                                                                                                                | INT          | Số tháng kể từ lần mua đầu tiên (0 = tháng đầu, 1 = tháng tiếp theo,...)                                 |
| revenue                                                                                                                                                                                     | FLOAT        | Doanh thu của đơn hàng (VNĐ) - có thể null nếu chưa hoàn tất                                             |
| Column                                                                                                                                                                                      | FLOAT        | Cột dự phòng / tính toán phụ (chưa xác định rõ nghĩa nghiệp vụ)                                          |
