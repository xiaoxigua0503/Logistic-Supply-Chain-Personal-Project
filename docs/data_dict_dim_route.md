| BẢNG: DIM_ROUTE                                                                                                                                  |              |                                                                       |
|--------------------------------------------------------------------------------------------------------------------------------------------------|--------------|-----------------------------------------------------------------------|
| Bảng chiều (Dimension) lưu thông tin các tuyến vận chuyển quốc tế mà công ty đang khai thác. Mỗi tuyến tương ứng với một quốc gia/tiền tệ nguồn. |              |                                                                       |
|                                                                                                                                                  |              |                                                                       |
| Tên Cột                                                                                                                                          | Kiểu Dữ Liệu | Mô tả & Ý nghĩa Nghiệp vụ                                             |
| route_id                                                                                                                                         | INT          | Khoá chính - Mã định danh duy nhất của tuyến vận chuyển               |
| exchange_rate                                                                                                                                    | FLOAT        | Tỷ giá quy đổi từ tiền tệ nước ngoài sang VNĐ (ví dụ: 1 IDR = x VNĐ)  |
| name                                                                                                                                             | VARCHAR      | Tên viết tắt của đơn vị tiền tệ hoặc tuyến (IDR, JPY, USD, KRW,...)   |
| note                                                                                                                                             | VARCHAR      | Mô tả đầy đủ tên tuyến (ví dụ: Tuyến Indo, Tuyến Nhật,...)            |
| ship_time                                                                                                                                        | DATETIME     | Ước tính thời gian vận chuyển (ngày dự kiến hàng về)                  |
| unit_buying_price                                                                                                                                | INT          | Đơn giá mua vào (VNĐ/kg) - giá công ty trả để nhận hàng               |
| unit_deposit_price                                                                                                                               | INT          | Đơn giá thu của khách (VNĐ/kg) - giá niêm yết với khách hàng          |
| difference_rate                                                                                                                                  | FLOAT        | Chênh lệch tỷ giá giữa giá mua và giá bán (margin tỷ giá)             |
| is_update_auto                                                                                                                                   | BOOL         | Cờ xác định tỷ giá có được cập nhật tự động theo thị trường hay không |
| min_weight                                                                                                                                       | FLOAT        | Trọng lượng tối thiểu tính cước (kg) - dùng để tính phí ship nhỏ nhất |
