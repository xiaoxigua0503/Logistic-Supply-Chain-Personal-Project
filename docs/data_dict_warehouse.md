| BẢNG: FACT_WAREHOUSE                                                                                                                                                                                                   |              |                                                                         |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------|-------------------------------------------------------------------------|
| Bảng sự kiện trung tâm theo dõi hành trình vật lý của hàng hoá qua các kho. Mỗi bản ghi là một lô hàng trong kho, ghi nhận trọng lượng, thời gian lưu kho, phí vận chuyển phân bổ và doanh thu phân bổ theo loại hàng. |              |                                                                         |
|                                                                                                                                                                                                                        |              |                                                                         |
| Tên Cột                                                                                                                                                                                                                | Kiểu Dữ Liệu | Mô tả & Ý nghĩa Nghiệp vụ                                               |
| warehouse_id                                                                                                                                                                                                           | INT          | Khoá chính - Mã định danh duy nhất của lô hàng trong kho                |
| order_id                                                                                                                                                                                                               | INT          | Khoá ngoại tới customer_order - Đơn hàng sở hữu lô hàng này             |
| route_id                                                                                                                                                                                                               | INT          | Khoá ngoại tới dim_route - Tuyến vận chuyển của lô hàng                 |
| name                                                                                                                                                                                                                   | VARCHAR      | Mã tiền tệ/tuyến (IDR, JPY, USD,...)                                    |
| ngay_nhap_kho_nn                                                                                                                                                                                                       | DATETIME     | Ngày hàng nhập kho tại nước ngoài (kho trung chuyển)                    |
| ngay_xuat_kho_nn                                                                                                                                                                                                       | DATETIME     | Ngày hàng xuất kho nước ngoài (đã bay về)                               |
| ngay_nhap_kho_vn                                                                                                                                                                                                       | DATETIME     | Ngày hàng nhập kho tại Việt Nam                                         |
| ngay_giao_vnpost                                                                                                                                                                                                       | DATETIME     | Ngày bàn giao hàng cho VNPost để giao tới khách                         |
| net_weight                                                                                                                                                                                                             | FLOAT        | Trọng lượng thực tế của lô hàng (kg)                                    |
| price_ship                                                                                                                                                                                                             | INT          | Phí vận chuyển của toàn bộ đơn hàng (VNĐ)                               |
| min_weight                                                                                                                                                                                                             | FLOAT        | Trọng lượng tính cước tối thiểu của tuyến (kg)                          |
| final_weight                                                                                                                                                                                                           | FLOAT        | Trọng lượng tính phí cuối cùng = max(net_weight, min_weight)            |
| revenue_main                                                                                                                                                                                                           | FLOAT        | Doanh thu chính của toàn đơn (phí ship × final_weight, VNĐ)             |
| product_type_id                                                                                                                                                                                                        | INT          | Khoá ngoại tới product_type - Mã loại sản phẩm trong lô                 |
| product_type_name                                                                                                                                                                                                      | VARCHAR      | Tên danh mục sản phẩm trong lô hàng                                     |
| n_types                                                                                                                                                                                                                | INT          | Số lượng loại sản phẩm khác nhau trong cùng đơn hàng                    |
| share                                                                                                                                                                                                                  | FLOAT        | Tỷ lệ phân bổ doanh thu cho loại hàng này trong đơn (0-1)               |
| final_weight_allocated                                                                                                                                                                                                 | FLOAT        | Trọng lượng phân bổ cho loại hàng này = final_weight × share (kg)       |
| revenue_allocated                                                                                                                                                                                                      | FLOAT        | Doanh thu phân bổ cho loại hàng này = revenue_main × share (VNĐ)        |
| date                                                                                                                                                                                                                   | DATE         | Ngày nhập kho nước ngoài (chuẩn hoá, dùng cho phân tích theo thời gian) |
| customer_id                                                                                                                                                                                                            | INT          | Mã khách hàng sở hữu lô hàng (khoá ngoại)                               |
| thời gian nằm kho nn                                                                                                                                                                                                   | FLOAT        | Số ngày hàng lưu tại kho nước ngoài (ngay_xuat - ngay_nhap kho nn)      |
| thời gian nằm kho vn                                                                                                                                                                                                   | FLOAT        | Số ngày hàng lưu tại kho Việt Nam (ngay_giao_vnpost - ngay_nhap_kho_vn) |
| thời gian bay                                                                                                                                                                                                          | FLOAT        | Số ngày vận chuyển đường bay (ngay_nhap_kho_vn - ngay_xuat_kho_nn)      |
